# Image Deteriorator

**Инструмент для ухудшения качества изображений снежного барса**

Простая утилита, которая создаёт ухудшенные копии высококачественных фото (профессиональные снимки, зоопарк и т.д.), чтобы модель YOLO лучше работала на реальных данных с фотоловушек.


## 🎯 Зачем это нужно

Высококачественные изображения слишком чистые. Реальные камеры дают шум, размытие, артефакты JPEG и потерю деталей. Этот инструмент применяет к каждому изображению случайный набор деградаций и сохраняет результат с тем же именем и структурой папок.


## ✨ Что делает pipeline

- Изменение яркости и контраста
- Сдвиг цвета и насыщенности
- JPEG-сжатие
- Атмосферный туман
- Хроматическая аберрация и оптические искажения
- Downscale и Defocus
- Разные виды шума (ISO, Gauss, Shot Noise)
- Иногда преобразование в серый цвет (симуляция ночного режма)


## 🚀 Установка

~~~bash
git clone https://github.com/Kakoy-Nikakoy-Corp/image-deteriorator.git
cd image-deteriorator
uv sync
~~~


## 📖 Как использовать

~~~bash
python deteriorate.py <input_dir> <output_dir>
~~~

Пример:

~~~bash
python deteriorate.py data/high_quality data/worsened
~~~
